# Authentication Enumeration

- Password policies can give valuable insights into the complexity of the passwords used in the application.
- The attacker then can guage the potential complexity of the passwords and tailor their strategy accordingly.

## Common places to enumerate

- Registration pages
- Password reset features
- verbose errors
- Data breach information

## Understanding verbose errors

- These detailed error messages are invaluable during the debugging process, helping developers to see what went wrong.
- These verbose errors can give insights about :
  - Internal paths
  - Database details
  - User information
